FrontPage di Microsoft è un sistema di sviluppo web integrato che consiste in un editor HTML, uno strumento di gestione del sito, e un server Web personalizzato. FrontPage fornisce una panoramica grafica del tuo sito, la visualizzazione di collegamenti mancanti o rotti con icone "strappate documento". L'editor HTML all'interno di FrontPage consente di creare documenti HTML complesso con le forme e le mappe immagine cliccabili. Fornisce inoltre il controllo della sintassi HTML, tra cui l'esame degli URL locale e remoto per assicurarsi che essi pioint di documenti validi. FrontPage rende le forme e le mappe immagine cliccabili "dal vivo", che consente di testare le pagine in locale senza installarli sul server.
